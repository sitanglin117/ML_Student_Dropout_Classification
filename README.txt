Machine Learning Project (Classification)

-----------------------------------------------------
1. Environment
-----------------------------------------------------
- Python version: 3.10
- Tested on: Python 3.10.x
- Operating systems: Linux / macOS / Windows

-----------------------------------------------------
2. Setup Instructions
-----------------------------------------------------
1. Create a virtual environment (recommended):
   python3.10 -m venv venv

2. Activate the virtual environment:
   - Linux / macOS:
     source venv/bin/activate
   - Windows:
     venv\Scripts\activate

3. Install required packages:
   pip install -r requirements.txt

-----------------------------------------------------
3. Reproducibility
-----------------------------------------------------
- Random seeds are fixed to ensure reproducible results.
- All preprocessing steps are included in the code.
- Relative paths are used for data loading.

-----------------------------------------------------
4. Notes
-----------------------------------------------------
- If the code does not run as expected, please ensure that the correct
  Python version and required packages are installed.
- Datasets are loaded locally and must be placed in the 'data/' directory.
